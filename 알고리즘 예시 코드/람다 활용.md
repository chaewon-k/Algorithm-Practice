# 람다 활용

## sort

```python
import re

def solution(files):
    answer = []
    table = []
    for file in files:
        table.append(re.split(r"([0-9]+)", file))
        
    ## 두 가지의 기준으로 정렬할 때
    sort_table = sorted(table, key = lambda x: (x[0].lower(), int(x[1])))

    for i in sort_table:
        answer.append(''.join(i))

    return answer
```
